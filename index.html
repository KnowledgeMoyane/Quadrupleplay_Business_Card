1. IPO – Input · Process · Output

IPO is one of the most common theory questions. You must be able to identify the three parts from a problem description.

Exam favourite
PartMeaning
InputData entered by the user (or read from a file)
ProcessCalculations or operations performed
OutputResult displayed to the user

Example: discount calculator

Problem: A program asks the user for a price, calculates a 10% discount, and shows the final price.

Inputprice
Processdiscount = price * 0.10 and final_price = price - discount
Outputfinal_price
📌 Exam tip: When asked for IPO, do not write code. Write the variables or values used for input, processing, and output.

2. Variable Naming Rules

✅ Valid variable names

  • student_name — letters and underscore
  • totalMarks — starts with a letter
  • _count — can start with underscore
  • age2 — numbers after first char

❌ Invalid variable names

  • 2age — starts with a number
  • total marks — contains space
  • class — Python keyword
  • student-name — contains hyphen
Remember: Must begin with letter or underscore · no spaces · no special symbols except _ · cannot use Python keywords.

3. Finding and Correcting Errors

Example 1: missing colon

# Incorrect
if price > 100
    discount = price * 0.1

# Correct
if price > 100:
    discount = price * 0.1

Example 2: wrong indentation

# Incorrect
if price > 100:
discount = price * 0.1

# Correct
if price > 100:
    discount = price * 0.1

Example 3: using = instead of ==

# Incorrect (logic error)
if discount = 10:
    print("standard discount")

# Correct
if discount == 10:
    print("standard discount")
🔍 Quick exam checklist:
✓ colon after if, else, for, while · ✓ indentation · ✓ brackets () · ✓ == for comparisons

4. For Loops

Understanding the output

for i in range(5):
    print(i)   # 0 1 2 3 4

range(5) means: start at 0, stop before 5.

CodeOutput
range(3)0 1 2
range(1, 4)1 2 3
range(2, 10, 2)2 4 6 8
range(10, 0, -2)10 8 6 4 2

Modify the loop

# Original: prints 0 1 2 3 4
for i in range(5):
    print(i)

# Modified to print 1–5
for i in range(1, 6):
    print(i)

# Modified to print even numbers 2 4 6 8 10
for i in range(2, 11, 2):
    print(i)

Exam-style question: What is the output of for i in range(3, 8): print(i) ?

Answer: 3 4 5 6 7

6. Functions

Basic function

Complete the function

# Question: write a function that returns the square of a number
def square(x):
    return x * x

print(square(5))   # 25

Calling a function inside another function

def multiply(a, b):
    return a * b

def calculate():
    result = multiply(4, 5)
    print(result)

calculate()   # 20

7. Text Files

Step 1: Open a file

file = open("marks.txt", "r")   # r = read mode

Step 2: Read values

If marks.txt contains:

Alice,80
Bob,70
for line in file:
    name, mark = line.strip().split(",")
    print(name, int(mark) + 5)

Output: Alice 85 · Bob 75

Step 3: Update the file

file = open("marks.txt", "r")
lines = file.readlines()
file.close()

updated = []
for line in lines:
    name, mark = line.strip().split(",")
    new_mark = int(mark) + 5
    updated.append(f"{name},{new_mark}\n")

file = open("marks.txt", "w")
file.writelines(updated)
file.close()
ModeMeaning
rRead
wWrite (overwrite)
aAppend

Full exam-style example

Question: Read prices from a file, apply 10% discount, save updated prices.

File contents (prices.txt):

Bread,20
Milk,15
file = open("prices.txt", "r")
lines = file.readlines()
file.close()

updated = []
for line in lines:
    item, price = line.strip().split(",")
    new_price = float(price) * 0.9
    updated.append(f"{item},{new_price}\n")

file = open("prices.txt", "w")
file.writelines(updated)
file.close()

# Updated file:
# Bread,18.0
# Milk,13.5

  • Fix : and indentation errors

  • Nested if/else discount logic

This is the exact pattern used in most file-processing exam questions.

⚡ A final 30-second exam strategy

  • • IPO question? Identify variables only.
  • • Loop question? Write out the numbers one by one.
  • • Nested if question? Check the outer condition first.
  • • Function question? Make sure there is a return if a value is needed.
  • • File question? open → readlines → split → calculate → writelines.
