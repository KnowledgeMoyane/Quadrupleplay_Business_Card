1. Boolean Expressions

Produces only True or False. Used for decisions.

5 > 3 # True 10 < 4 # False 8 == 8 # True age = 20 if age >= 18: # condition is a Boolean expression print("You may vote")
example 5 > 3 → True
example 10 < 4 → False

3–6. Decision Structures: if, if-else, if-elif-else

# if age = 20 if age >= 18: print("Adult") # if-else mark = 45 if mark >= 50: print("Pass") else: print("Fail") # if-elif-else mark = 72 if mark >= 75: print("Distinction") elif mark >= 50: print("Pass") else: print("Fail")
⚠️ Common errors: missing colon (:) · wrong indentation · using = instead of ==.

7. Identifying Errors in Decision Structures

❌ wrong if age > 18 (missing colon)
✅ correct if age > 18:
❌ wrong indentation: if age > 18:\nprint("Adult")
✅ correct indent the block
❌ wrong if age = 18: (assignment instead of comparison)
✅ correct if age == 18:
