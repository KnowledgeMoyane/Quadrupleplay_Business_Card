1. Internet & WWW

🌐 Internet — global network of computers.
Roads = Internet · cars = data · destinations = websites.
Services: email, banking, social media, streaming, gaming, cloud.
📄 WWW — collection of web pages & sites running on the Internet.
Internet = network · WWW = websites
InternetWWW
physical networkcollection of websites
infrastructureservice running on Internet
exists without websitesdepends on Internet
📌 How a web page loads: URL → DNS finds IP → browser requests → server returns HTML/CSS/JS/images → browser displays.

2. HTTP & HTTPS

HTTP = HyperText Transfer Protocol · transfers web pages. http://example.com
HTTPS = HyperText Transfer Protocol Secure · encrypted. https://example.com
HTTPHTTPS
not securesecure
data visibledata encrypted
no SSLuses SSL/TLS certificate
less trustedmore trusted
✅ For online banking → always use HTTPS. Protects passwords, banking details, prevents hackers.

3. Accessibility

Make websites usable for everyone — blind, deaf, colour-blind, elderly.

alt text <img src="dog.jpg" alt="Brown dog running"> · screen readers describe images.
headings use <h1> to <h6> in order.
contrast black on white (good) · yellow on white (bad).
semantic HTML <header> <nav> <main> <footer> improves understanding.

4. Responsive Design

Adapts to phones, tablets, laptops, desktops.
<meta name="viewport" content="width=device-width, initial-scale=1.0">
✅ better UX · mobile friendly · faster navigation

5. HTML Document Structure

<!DOCTYPE html> <html> <head> <title>My Website</title> </head> <body> <!-- content --> </body> </html>
<!DOCTYPE html> → HTML5
<html> → root
<head> → metadata
<body> → visible content

6. Meta Tags

<meta charset="UTF-8"> <meta name="description" content="Computer Science Notes"> <meta name="viewport" content="width=device-width, initial-scale=1.0">

7. Headings, Paragraphs & Text

<h1> to <h6> · <p> paragraph · <br> line break
bold: <strong> · italic: <em>

8. Hyperlinks

<a href="https://google.com">Google</a> <!-- external --> <a href="about.html">About</a> <!-- internal --> <a href="mailto:test@gmail.com">Email Us</a>

<main> main content · <section> section
<article> independent · <footer> bottom

12–20. CSS Basics · Selectors · Colours · Fonts · Borders · Spacing · Box Model · Flexbox · Grid

CSS syntax: selector { property: value; }
p { color: blue; }
Selectors: element p · class .note · id #title
Colours: named red · hex #FF0000 · background background-color: yellow;
Fonts: font-family: Arial; · font-size:20px; · font-style:italic;
Borders: border:2px solid black;
Spacing: padding (inside) · margin (outside)
Box Model: Margin → Border → Padding → Content
Flexbox: display:flex; · justify-content:center; · align-items:center;
Grid: display:grid; grid-template-columns:1fr 1fr; · 2D layout
Flexbox = one dimension · Grid = two dimensions (rows & columns)

21. HTML Tables

<table> <tr><th>Name</th><th>Age</th></tr> <tr><td>John</td><td>20</td></tr> </table>
<table> · <tr> row · <th> heading · <td> data · <caption> title
CSS hover: tr:hover { background-color:yellow; }

22–28. Forms · Input types · Radio · Checkbox · Dropdown · Textarea · Validation

Form: <form action="save.php" method="POST">
Input types: text, email, number, date, password
Radio: <input type="radio" name="gender"> (one choice)
Checkbox: <input type="checkbox"> (multiple)
Dropdown: <select><option>SA</option></select>
Textarea: <textarea></textarea> for large text
Validation: required · type="email" · min="1" max="100"

29–34. JavaScript Variables · Functions · prompt/alert · Decisions · Validation · Error Handling

// variable let name = "John"; let age = 20; let passed = true; // function function greet() { alert("Hello"); } greet(); // prompt & alert let name = prompt("Enter your name"); alert("Welcome"); // decision if (age >= 18) { alert("Adult"); } else if (age >= 13) { alert("Teen"); } else { alert("Minor"); } // validation if (age == "") { alert("Age required"); } if (isNaN(age)) { alert("Numbers only"); }

35–37. jQuery · Events · Methods

jQuery = JavaScript library · easier coding.
$("p") selects all paragraphs · $("#title") · $(".box")
Click: $("#btn").click(function(){ alert("Clicked"); });
Methods: hide() · show() · fadeIn() · text("Hello") · css("color","red")

38. Integration (HTML + CSS + JS + jQuery)

HTML = structure · CSS = appearance · JavaScript = functionality · jQuery = easier JavaScript
