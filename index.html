1. What is HTML?

HTML (HyperText Markup Language) is the standard language used to create the structure and content of webpages. Every element you see — headings, paragraphs, images, buttons, links — exists because an HTML tag placed it there.

HTML does not make things look pretty. It only defines what exists and in what order.

Minimal HTML page

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello World</h1>
    <p>This is my first webpage.</p>
  </body>
</html>

2. What is CSS?

CSS (Cascading Style Sheets) is the language used to style HTML elements — controlling colors, fonts, sizes, spacing, layout, animations, and more. CSS cannot create content; it only changes how existing HTML elements appear.

CSS is called "cascading" because when multiple style rules target the same element, they follow a specific order of priority.

CSS that styles the page above

body {
  font-family: Arial, sans-serif;
  background-color: #f0f0f0;
  margin: 40px;
}
h1 {
  color: navy;
  font-size: 36px;
}
p {
  color: #444;
  line-height: 1.6;
}

3. The core difference

Analogy: HTML is the architecture — front door, rooms, windows. CSS is the interior design — paint, curtains, furniture. You need both, but they do completely different jobs.

HTML answers: What is on the page? Is this a heading or a paragraph? What text? What order?

CSS answers: What color? How large? Where is it positioned? Padding? Border? Hover effect?

4. HTML syntax explained

HTML is made up of elements. Each element consists of an opening tag, content, and a closing tag. Some tags are self-closing.

<h1>Hello World!</h1>
<a href="https://example.com">Visit us</a>

<img src="photo.jpg"> · <br> · <hr> · <input type="text"> are self-closing.

7. CSS syntax explained

A CSS rule has a selector, a property, and a value.

h1 {
  color: navy;
  font-size: 32px;
}

Three types of CSS selector

/* 1. Element selector */
p { color: gray; }

/* 2. Class selector */
.highlight { background: yellow; }

/* 3. ID selector */
#header { background: navy; }

/* combined, descendant, pseudo */
h1, h2, h3 { font-family: Georgia; }
.card p { color: #555; }
.btn:hover { background: darkblue; }

8. Linking CSS to HTML

External (best practice):

<link rel="stylesheet" href="styles.css">

Internal (single page):

<style> h1 { color: navy; } </style>

Inline (avoid): <h1 style="color: navy;">

✅ c) <link rel="stylesheet" href="styles.css">

✅ CSS

identify

multiple choice
